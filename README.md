# Boba POS
A console-based boba tea POS system, featuring 3 separate menus (milk tea type, size, and toppings) and a receipt generator. Upon execution, the program will print one menu then prompt the user to input their choice. At the end of the program, a receipt of the order will be printed.

## Versions
**Boba_POS_original.py** - Original version using parallel lists for menus' items and prices, prompting user for an integer input

**Boba_POS.py** - Refactored version using dictionaries in place of parallel lists, now prompting user for choice name; also implemented exception handling.
